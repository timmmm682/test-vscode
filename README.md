# 🌌 TRANCE STATE

An immersive, multi-page digital sanctuary designed to induce focus and flow through hypnotic visuals and neon aesthetics.

## 🚀 Live Demo
Experience the state of flow: **[https://timmmm682.github.io/test-vscode/](https://timmmm682.github.io/test-vscode/)**

## ✨ Features
- **Hypnotic Visualizer**: A custom-built Canvas API particle system that reacts to your movement and changes pace across different pages.
- **Neon Aesthetic**: High-contrast, glowing UI elements with a deep space color palette.
- **Glitch Effects**: Dynamic CSS-powered text glitching for a futuristic cyber-trance vibe.
- **Interactive Cursor**: A custom glowing ring cursor that interacts with the background particles.
- **Glassmorphism UI**: Modern, semi-transparent layouts that let the background visuals bleed through.
- **Multi-Page Journey**:
  - **Home**: The portal to the experience.
  - **Experience**: Maximum intensity visuals and minimal UI.
  - **About**: Discover the philosophy of the trance state.
  - **Connect**: A direct line to the collective.

## 🛠️ Tech Stack
- **HTML5**: Semantic structure for accessibility and SEO.
- **CSS3**: Advanced animations, keyframes, and custom properties (variables).
- **JavaScript (ES6+)**: Custom particle physics engine and interactivity.
- **Google Fonts**: Orbitron for that signature futuristic look.

## 🎨 Customization
The project uses CSS variables for easy theme changes:
```css
:root {
    --neon-cyan: #0ff;
    --neon-magenta: #f0f;
    --bg-color: #050014;
}
```
